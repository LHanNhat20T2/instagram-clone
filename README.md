# INSTAGRAM CLONE 
## FEATURES

*User*
- Signup
- Login
- Logout
- Get, edit profile
- Suggested profile
- Folow or unfollow friend
  
*Post*
- Add, get, delete post
- Like or dislike post
- Comment post
- Bookmark post

*Messsage*
- Chat
  
## TECHNOLOGIES
### Backend:
- Nodejs, expressjs, jwt, cloudinary, socket io, cookie
### Frontend:
- Reactjs, react-router-dom, react-hook-form, yup,redux toolkit, Axios, taildwincss, shadcn
### Database: 
- Mongoose db
